RPM regulation


    Run demo
    (To run this application Java Runtime Environment is necessary. www.oracle.com)
One of the commonly used ways to set the operating point of the pump is RPM regulation. They use a frequency regulator to change the RPM of the machine, to achive a different operating point, but this leads, to a different characteristics.

If we know the given flow volume, and the characteristics of the pipes, we can calculate the new RPM.

This RPM can be calculated using a parabola of affinity. The parabola starts at the origo, intersects the new point of operation and the original characteristics curve. The intersection of the parabola and the characteristics is the auxiliary point. We can use the rules of affinity between this point and the new operating point:
$$\frac{n_1}{n_2}=\frac{Q_1}{Q_2}$$ $$\left(\frac{n_1}{n_2}\right)^2=\frac{H_1}{H_2}$$ $$\left(\frac{n_1}{n_2}\right)^3=\frac{P_1}{P_2}$$
Index 1 is for the auxiliary pont, index 2 is for the operating point.
