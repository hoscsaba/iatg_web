Gas compression


    Run demo
    (To run this application Java Runtime Environment is necessary. www.oracle.com)
During the compression of ideal gases and liquids the energy required for the state change can be calculated by $P=\dot{m} Y$ in which $\dot{m}[kg/s]$ is the mass flow, and $Y[J/kg]$ is the specific work. (For easily compressable substances - eg. air - mass flow is equal on the input and output sides of the compressor, but volume flow is not $Q=\dot{m}/\rho$.) Specific work can be calculated as such: $$Y=\int_{p_1}^{p_2} v dp=\int_{p_1}^{p_2} \frac{1}{\rho(p)} dp$$ To calculate the work, we have to be familiar with the state change, the $\rho(p)$ function. Let's see the easiest one first!
  • When the density is constant (liquids) $$Y=\frac{1}{\rho}\int_{p_1}^{p_2}1 dp=\frac{p_2-p_1}{\rho}$$
  • For Isothermal process $T=$constant, so for ideal gas ($p/\rho=RT$) $$Y=\int_{p_1}^{p_2}\frac{RT}{p} dp=RT\int_{p_1}^{p_2}\frac{1}{p} dp=RT\ln \frac{p_2}{p_1}$$
  • For Isentropic process $p_1/\rho_1^\kappa=p/\rho^\kappa=$constant $$Y=\frac{\kappa}{\kappa-1}RT_1 \left( \left(\frac{p_2}{p_1} \right)^{\frac{\kappa-1}{\kappa}}-1\right)=\frac{\kappa R}{\kappa-1} \left(T_2-T_1\right)=h_2-h_1,$$ where $h[J/kg]$ is the enthalpy of the substance.
  • For Politropic process $p_1/\rho_1^n=p/\rho^n=$constant, so the equations are formally identical to the isentropic process: $$Y=\frac{n}{n-1}RT_1 \left( \left(\frac{p_2}{p_1} \right)^{\frac{n-1}{n}}-1\right)=\frac{n R}{n-1} \left(T_2-T_1\right)$$ The $n$ politropic index usually comes from measurements. When $n\rightarrow 1$ we get an isothermal process, when $n=\kappa$ we get an isentropic process.
Lastly, the change in entrophy of an ideal gas can be calculated by this equation: $$s_2-s_1=c_V \ln \frac{T_2}{T_1}+R\ln \frac{\rho_1}{\rho2}$$ $\clubsuit$
